## 📝 如何认领任务

1. 在 GitHub 上找到您感兴趣的 Issue
2. 在 Issue 评论区留言 "I'll work on this" 或 "我来处理这个"
3. 开始工作，完成后提交 Pull Request
4. 在 PR 中引用对应的 Issue 号（如 Fixes #20）
